# TACS-UTN-2021

## ¿Cómo levantar el TP?
Desde el directorio raiz, ejecutar el comando

`$ docker-compose up --build -d`

#####Para buildear y levantar backend y base de datos. 

#####Para builder el .jar del backend:

mvn package  -DskipTests=true
